Uber Data Analysis Dashboard (Power BI)
Overview

This project is a Power BI dashboard built to analyze Uber ride data, focusing on bookings, revenue, trip distance, and payment methods.

Features
Total bookings and revenue analysis
Average trip distance
Ride status distribution
Payment method insights
Interactive filters and slicers
Tools Used
Power BI
DAX
CSV Dataset
